# Gridlock 2.0: AutoTraffic Command Center AI

An automated vision intelligence system optimized for complex urban traffic modeling, edge violation tracking, and cryptographic digital citation (E-Challan) generation.

## 🛠️ Features Implemented
- **Image Preprocessing Layer:** Adaptive handling for low light, rainfall, shadows, and headlight lens flares.
- **Unified Object Tracking Tensors:** Specialized deep detection modeling isolating riders, drivers, helmets, and stop-line cross boundaries.
- **License Plate Recognition (OCR):** Context-aware regional Indian registration string translation engine.
- **Evidence Management Workspace:** Real-time generation of cryptographic legal documentation PDFs directly tied to absolute localized timestamps.
